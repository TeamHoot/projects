## 🚀 GitHub Pages 배포 방법

1. 이 레포지토리를 GitHub에 올립니다
2. **Settings** → **Pages** 탭으로 이동
3. **Branch**: `main` / **Folder**: `/ (root)` 선택 후 **Save**
4. 1~2분 후 위 주소로 접속 가능

---

## 📁 파일 구조

```
eplink-site/
├── index.html   # 메인 페이지 (모든 리소스 포함)
└── README.md
```

> 모든 이미지와 아이콘은 base64로 HTML에 임베드되어 있어
> 별도 파일 없이 `index.html` 하나로 완전히 동작합니다.
